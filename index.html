Actualizar Sistema

Primero actualiza la lista de paquetes y el sistema:

sudo apt update && sudo apt upgrade -y

Instalar Apache

Instala el servidor web Apache2:

sudo apt install apache2 -y

Verificar Instalación

Comprueba que Apache se instaló correctamente:

sudo systemctl status apache2

Comandos de Control del Servicio

sudo systemctl start apache2
sudo systemctl stop apache2
sudo systemctl restart apache2
sudo systemctl enable apache2

Configuración del Firewall (UFW)

Ubuntu incluye UFW (Uncomplicated Firewall). Configura las reglas para permitir tráfico web:

Comandos de Configuración UFW

sudo ufw enable
sudo ufw allow 'Apache Full'
sudo ufw allow ssh
sudo ufw status verbose

Estructura de Directorios de Apache

Directorios Principales

/var/www/html Document root (archivos del sitio)
/etc/apache2 Archivos de configuración
/etc/apache2/sites-available Configuraciones de sitios disponibles
/etc/apache2/sites-enabled Sitios activados (enlaces simbólicos)
/var/log/apache2 Archivos de log

Crear tu Primer Sitio Web

sudo mkdir -p /var/www/misitio
sudo chown -R $USER:$USER /var/www/misitio
sudo chmod -R 755 /var/www/misitio

Configuración de Virtual Host

Crea un archivo de configuración para tu sitio:

sudo nano /etc/apache2/sites-available/misitio.conf
# Configuración básica de Virtual Host
<VirtualHost *:80>
    ServerAdmin admin@misitio.com
    ServerName misitio.com
    ServerAlias www.misitio.com
    DocumentRoot /var/www/misitio
    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined

    <Directory /var/www/misitio>
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>

Activar y Desactivar Sitios

sudo a2ensite misitio.conf
sudo a2dissite 000-default.conf
sudo systemctl reload apache2
